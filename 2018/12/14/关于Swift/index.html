关于Swift

Swift简介

Swift

  • 中文含义：敏捷、快速
  • 新一代苹果平台（ios/macOS/watchOS/tvOS）的app及游戏开发语言
  • 语法与JavaScript、C#等语法类似、上手快

历史和未来

  • 2014年6月推出第一个版本，由苹果员工Chris历经4年发明
  • Swift已经开源，计划逐渐替代Objective C语言，成为ios开发的主流语言
  • 长期致力于成为未来20年内运行到每一部智能设备上

Swift的创新

  • 于引入Optional（可选类型）

  • 支持传统的“面向对象编程”，提升至“面向协议编程”；即由“类”到“值”，提倡使用结构体取代类（class）,”遵守协议”取代“继承”

常量

  • 预期范围内恒定不变的量

  • let 常量名

  • 常量名可以是任意文字 长度不限

变量

  • 常量与变量在一定条件下可以互相转换

  • 常量是变量的一种

值的类型和类型推断

  • 值的类型

  • Swift 有类型推断，类型可以省略不写

    浮点型（小数）

Swift中默认浮点型是Double（双精度）

类型安全

  • 变量一旦定义，其类型不可更改，即：不能给变量一个类型不同的值

布尔型

  • 布尔型：Bool ,表示逻辑上的“真”或“假”

  • 只有ture 和 false

Tuple

  • 定义变量的一个组合

Optional

  • 可选类型：代表变量可能有值的情况

    如：用户资料的选填部分

  • 形式： var 变量 ：类型？，默认是无值（nil）

基础操作符

  • 操作几个对象，被称为几元操作符

一元操作符

二元操作符

赋值操作符

  • 用等号右边的值来更新左边的值

数学操作符

+、-、*、/、%

组合赋值操作符

  • 赋值与数学操作符的组合，比如 +=

比较操作符

  • < ,<=,>,>=，结果是true或false

逻辑操作符

  • Bool值的组合
    • 与：每项全部为true，整体为true
    • 或：有一项为true，整体为true
    • 非：对Bool值取反

字符和字符串

  • String是字符串类型，Character是字符类型
  • 字符串是有序的字符集合，或者叫文本，如“我今天真可爱”
  • 通过字符串插值可以合成一个长字符串

  • 空字符串：用“”字面量

    • “ ”不是空字符

  • 可以对一个字符串的characters属性进行循环，来访问其中单个字符

  • 连接字符和字符串
    • 用 +
  • 向字符串添加字符，用append（单个字符）
  • 字符串插值：组合常量/变量/字面量/表达式成一个长字符串

集合类型

  • 一组同类型的值的组合，根据组合的整体特性可以分为：

    • 有序可重复 - 数组（Array）
    • 无序不重复 - Set
    • 无序可重复，但每个值有唯一的键（Key）- 字典（Dictionary）

  • 批量处理集合中的元素，可以使用for in 循环

数组

  • 定义：Array<类型>或[类型]
  • 创建一个有默认值的数组
  • 创建一个有序范围的Int数组，Array（起始值…终止值）
  • 用数组字面量来创建数组：[值1，2，3，4…]
  • 元素计数:count，空否：isEmpty
  • 添加：1. append，添加一个同类型数组 +=
  • 获取元素：数组名[索引]，索引总是从零开始
  • 插入：insert
  • 移除：remove

集合

  • 适合存储具有唯一性的数据
  • 定义：Set<元素类型> ，（无序）
  • 用数组字面量创建集合
    1. 元素计数:count，空否：isEmpty
    2. 插入：insert
    3. 移除：remove
  1. 是否包含某元素：contains
  2. 转换为数组：sorted
  1. 集合间的运算：交差并补
    • 交集：intersection
    • 差集：subtract
    • 并集：union
    • 补集：SymmetricDifferent

字典

  • 定义：Dictionary<键类型，值类型>，或[键类型：值类型]
  • 同字典字面量来创建字典，键与值之间用冒号隔开
  • 创建数组
    1. 元素计数:count，空否：isEmpty
    2. 添加或更新：字典变量[键] = 值
    3. 获取：字符变量[键]
    4. 移除：字符变量[键] = nil

  1. 序列中每项都参与，for与in之间加一个代替每项的名
  1. 只需循环固定次数，可以省略项名

if-else

  • 循环中的判断

    continue结束本次循环，break结束所有

函数

  • 定义函数：func 函数名（参数1：类型，参数2：类型） -> 返回结果的类型 {执行语句}
  • 调用：var 变量名称 = 函数名（变量1，变量2…）

闭包 Closure

  • 闭包是一种函数的简写形式，省去函数名，把参数和返回值放入花括号内

枚举

  • 计数、列举；
